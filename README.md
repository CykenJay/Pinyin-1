# Pinyin
一个简单的基于js的拼音输入法
# web输入法
 只使用了原生js代码，单文件无依赖，总大小仅300KB
 联想几乎没有（用的上古时期智能ABC的字典），能用（也仅仅是能用）。
 目的是让新安装Linux的同学在没有安装输入法之前有一个能输入中文的临时用品。
 
 有图形浏览器的地方就能用，真 跨平台。
